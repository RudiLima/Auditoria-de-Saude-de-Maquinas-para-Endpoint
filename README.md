🔍 O que o relatório coleta
Este relatório realiza uma varredura básica na máquina e coleta informações essenciais para análise de saúde do ambiente:
🖥️ Informações do Sistema

Nome do computador
Modelo e fabricante
Número de série
Versão da BIOS

⚙️ Hardware

Modelo do processador (CPU)
Temperatura da CPU (quando disponível)
Quantidade total de memória RAM

💿 Armazenamento

Espaço total e livre de cada disco
Percentual de uso
Tipo de disco (SSD/HDD)
Status SMART do disco

🔐 Segurança

Status do BitLocker
Status do Microsoft Defender

🔄 Atualizações

Quantidade de atualizações do Windows pendentes

📦 Softwares Instalados

Lista de softwares instalados
Versão de cada aplicação
Status (quando identificado)

📊 Score de Saúde

Índice geral baseado em critérios como:

Espaço em disco
Atualizações pendentes
Segurança ativa
Condição do hardware
